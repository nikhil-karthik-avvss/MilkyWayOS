# MilkyWayOS

A custom Linux distribution built from scratch on Debian 12.

## Components
- **Starlight** — Custom display server (DRM/KMS + evdev)
- **Nebula** — Custom desktop environment
- **Pulsar** — Custom terminal emulator
- **Orbit** — Custom app store

## Base
- Debian 12 (Bookworm), amd64
- Theme: Dark, deep space aesthetic
